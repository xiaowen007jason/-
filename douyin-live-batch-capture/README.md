# 小文抖音直播间批量截图（360 极速浏览器版）

这是一个本地浏览器扩展。它复用当前 360 极速浏览器中的抖音登录状态，不导出 Cookie，不切换账号环境。

品牌标识与工具界面版权归“小文”所有。© 2026 小文，本工具仅供授权业务使用。

## 适用流程

1. 把专用的 360 浏览器窗口放到竖屏副屏，并最大化。
2. 打开扩展，填写品牌名称，点击“打开直播搜索”。
3. 在抖音搜索页确认已切换到“直播”分类。
4. 再次打开扩展，点击“自动扫描当前页”。
5. 检查扫描出的直播间，取消不需要的账号。
6. 点击“开始批量截图”。运行期间不要操作这个 360 窗口。
7. 截图默认保存到浏览器下载目录下的 `直播间截图/品牌/日期/`。

文件名包含序号、品牌、直播间名称和完整截图时间，例如：

`01_谷雨_谷雨官方旗舰店_2026-07-17_10-35-20.png`

## 安装到 360 极速浏览器

1. 在地址栏打开 `chrome://extensions/`。
2. 打开右上角“开发者模式”。
3. 点击“加载已解压的扩展程序”。
4. 选择本文件夹 `douyin-live-batch-capture`。
5. 将“小文抖音直播间批量截图”固定到浏览器工具栏。

## 运行原则

- 只使用当前浏览器窗口和已有登录状态。
- 单标签页顺序访问，不并发打开大量直播间。
- 不自动点赞、关注、评论、下单或发送任何信息。
- 不访问抖音非公开接口。
- 遇到验证码或登录验证时停止任务，由用户手动处理。

## 当前版本说明

- 扩展会扫描当前搜索结果中已经加载出来的直播间，并自动滚动以加载更多结果。
- 0.2.0 起优先按直播视频的原始分辨率直接提取无损 PNG，不再把抖音的模糊背景层当作主画面。
- 如果原始帧受跨域限制，扩展会按竖屏比例、源分辨率和模糊滤镜等特征选择清晰主视频，临时铺满网页视口截图后恢复。
- 0.4.0 起默认使用“副屏实际像素”模式：将清晰主视频铺满竖屏网页后按副屏视口截图。若浏览器截图失败，会把540×960等直播源帧按2倍尺寸输出为1080×1920。
- 浏览器的 `captureVisibleTab` 接口要求 `activeTab` 或 `<all_urls>` 权限。为了让同一个标签页跨越搜索页和直播域名后仍能自动截图，清单声明了 `<all_urls>`；扩展的内容脚本仍只会注入抖音两个域名，代码不上传截图或浏览数据。
- 若抖音更新页面结构，直播间收集规则可能需要调整。
- 若这版 360 不支持 Manifest V3 或 `captureVisibleTab`，需要改为兼容清单或配套 Windows 截图程序。
